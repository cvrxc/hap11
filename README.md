# Book Tracker (Трекер прочитанных книг)
**Автор:** Анастасия Волкова
**Вариант:** Book Tracker
**Дата сдачи:** 04.05.2026
---
## Описание программы
*"Book Tracker-это простое GUI-приложение для учёта прочитанных книг: добавление, фильтрация по жанру и страницам, сохранение и загрузка данных в JSON. Всё работает на Python с Tkinter."*
---
## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `pip install matplotlib requests`
---
